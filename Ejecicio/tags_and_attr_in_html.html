Etiquetas y atributos html.

Etiquetas principales:

  • <html>:
    Se usa como inicio y cierre en una página html, es decir, lo que esté entre estas etiquetas el navegador lo tomará como código html. Solo se suele usar un atributo:
    • lang: Sirve para identificar qué lenguaje se va a utilizar en la página
  • <head>:
    Esta etiqueta va a continuación de "<html>" y sirve para insertar metadatos, scripts y documentos que nos sirvan para el funcionamiento de la página.
  • <body>:
    Dentro de esta etiqueta estará todo el código de la pagina que será mostrado por el navegador. No suele llevar atributos pero se podría usar:
    • style: Sirve para definir estilos del documento html (los que se definan aquí se tomarán como más importantes que los definidos por css a menos que lleven "!important")

Etiquetas dentro del head:

  • <title>: Entre estas etiquetas definirás el texto que aparecerá en la pestaña del navegador.
  • <base>: Indica la URL base de las URLs relativas de todo el documento.
  • <style>: Entre las etiquetas de estilo pondrás código css para dar un formato a la página.
  • <link>: Esta es una de las etiquetas que no tiene cierre y se utiliza para relacionar esta página con archivos externos.
  • <meta>: Recoge información sobre el documento, esto sirve para navegadores, motores de búsquedas y servicios Webs.
  • <script>: Se usa para la parte de cliente en los que se suele poner la ruta a un fichero javascript aunque tambien se puede poner código javascript entre estas etiquetas directamente. Tiene atributos como:
    • src: se utiliza para indicar la ruta de un archivo js.
    • type: para definir que tipo de script, los cuales son:
      • text/javascript
      • text/ecmascript
      • application/javascript
      • application/ecmascript
    • text: este atributo actúa como el "text-content".
    Otros atributos son:
    • defer
    • crossorigin
    • language
    • integrity
    • async
  • <noscript>: Ofrece un contenido alternativo para navegadores que tienen desactivado usar scripts o no soportan la tecnología de script.

Etiquetas del body:

  • Títulos:
    • <h1>: Solo puede haber un h1 en tóda la página. Ejemplo:

      Esto es un <h1>

    • <h2>: Pueden haber varios, pero no abusar. Ejemplo:

      Esto es un <h2>

    • <h3>:

      Esto es un <h3>

    • <h4>:

      Esto es un <h4>

    • <h5>:
      Esto es un <h5>
    • <h6>:
      Esto es un <h6>
  • Contenedores:
    • <div>: Es la etiqueta contenedora más utilizada hoy día, la cual sirve principalmente para la correcta maquetación de la página.
    • <section>: Define secciones en un documento, como capítulos, encabezados, pies de página o cualquier otra sección del documento.
    • Listas:
      • <ul>: Indica que se va a hacer una lista desordenada (todo este documento).
      • <ol>: Indica que se va a hacer una lista ordenada, la cual al verse en el navegador va a estar numerada de diversas formas.
      • <li>: Esta etiqueta se pondrá entre las dos etiquetas anteriormente definidas y marcan el contenido de dentro como elemento de la lista.
    • Tablas: con etiquetas de inicio y cierre <table>:
      • <tr>: secciona la tabla en partes.
      • <th>: utilizado para la tabe del "head" de una tabla.
      • <td>: define elementos dentro de esta.
    • <header>: define la cabecera del documento donde se pone el logo y elementos de navegación.
    • <footer>: es el llamado "pie de página" de las web.
    • <aside>: es un contenedor lateral.
    • <article>: es un articulo en sí de la web.
  • <p>: Es una etiqueta muy usada que define la información introducida como párrafo.
  • <br>: Hace un salto de líne e la página.
  • <span>: Esta etiqueta se utiliza para mostrar pequeños textos, sin tener que dividir las secciones del documento.
  • Etiquetas de formateo:
      Se recomienda utilizar:
    • <strong>: Pondrá la información de dentro en negrita.
    • <em>: Texto en itálica.
      • Ya que:
    • <strong>: negrita.
    • <strong>: itálica.
      • Han quedado obsoletas.
  • multimedia:
    • <img>: se utiliza para añadir imágenes y tiene los atributos:
      • src: para indicar la ruta de la imagen.
      • alt: para mostrar un texto alternativo.
    • <video>: Es utilizado para añadir videos a tu página web. Se incluyen dentro la etiqueta:
      • <source>: indicas el directorio por el atributo src y el tipo de video por type.
    • <audio>: Aquí indicas el tipo de audio y qué audio quieres añadir. Se incluyen dentro la etiqueta:
      • <source>: indicas el directorio por el atributo src y el tipo de audio por type.
  • Hiperlinks:
    • <a>: Dentro de pondrá información de hacia donde te dirige el link. Tiene de atributos los siguientes:
      • target: para definir donde abrir el enlace
      • download: descargar
  • <nav>: son las etiquetas de navegación.
